# Code for "Single-cell Type Annotation with Deep Learning in 265 normal cell types for humans"

## Overview:






bash.sh

full.py

Predicts each cell type in the test set.

prepare_gene_ind.py

separate.py
Separates data

train_deep.py
Trains the model on the training data.
